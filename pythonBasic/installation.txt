Step 1: Install Python
Download Python

Go Python official website:
https://www.python.org/downloads/

Important during installation:
Tick “Add Python to PATH”
Click Install Now

Step 2: Check Installation
Open Command Prompt (CMD) and type:
Bash
python --version

Output example:
Python 3.x.x
